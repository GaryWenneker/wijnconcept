# Storybook Addon Install

```sh
npm install mdx-embed storybook-addon-mdx-embed --save
```

### Setup

Add `storybook-addon-mdx-embed` to your `addons` array in `main.js`

```javascript
// main.js
module.exports = {
 ...
  addons: ['storybook-addon-mdx-embed']
};

```
